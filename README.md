# ROS Tutorial Package
This package is written for self learning, reference and beginner guide to ROS. It is written in ROS Melodic Ubuntu 18. [For Mum!]

### Installation
create catkin_ws on home directory
```
$ mkdir -p ws_catkin/src 
$ cd ws_catkin
$ catkin_make
$ echo "source ~/ws_catkin/devel/setup.bash" >> ~/.bashrc
```
